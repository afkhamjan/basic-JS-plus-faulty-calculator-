# JavaScript Basics for Front-End Development

This repository contains a **basic introduction to JavaScript** focused on **front-end development**.  
It is created to understand JavaScript fundamentals step by step, starting from setup to core programming concepts and practice problems.

---

##  Topics Covered

### 1. JavaScript Setup
- How to set up JavaScript for front-end development
- Running JavaScript in the browser
- Using the console for output

### 2. Variables & Data Types
- `var`, `let`, and `const`
- Understanding variable scope
- Basic data types
- `console.log()` for debugging and output

### 3. Conditional Statements
- `if` statement
- `if-else`
- `if-else-if`
- Decision making in JavaScript

### 4. Operators
- Assignment operators
- Comparison operators
- Logical operators
- Ternary operator

### 5. Switch Statements
- `switch` case
- `break` statement
- Default cases

### 6. Loops
- `for` loop
- `while` loop
- `do-while` loop
- Loop-based logic building

### 7. Functions
- Function declaration
- Function calling
- Parameters and return values
- Reusability of code
## Practice Programs

To strengthen understanding, the following programs were implemented:

- Age Calculator
- Maximum and Minimum number finder
- Even and Odd number checker
- Faulty Calculator (using `eval`)

 Purpose of This Repository

- Build a strong foundation in JavaScript
- Understand core programming concepts
- Prepare for advanced JavaScript and front-end frameworks
- Practice problem-solving using JavaScript
